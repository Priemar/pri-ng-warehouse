pri-ng-warehouse Documentation
Github

pri-ng-warehouse is an angular module based on localforage, which allows you to simply store data in the browsers store(s).

  • Simple API
  • Same API for different storage types
  • Storage types: indexedDB, websql, localstorage, inmemory
  • Fallback to supported storage type if configured storage is not available
  • Use mutliple storages at the same time
  • Compatible with SSR
